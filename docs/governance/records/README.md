# Geospatial Governance Records

Use this folder for assurance and risk-governance evidence.

## Required subfolders

- `geospatial-assurance/`
- `geospatial-risk/`

## Required periodic artifacts

Monthly:

- access and audit review output
- model and threshold review memo
- risk-board snapshot and actions

Quarterly:

- DR drill report
- independent assurance bundle summary
- compliance and export control review

## Control requirements

- Every file must include owner and date.
- Exceptions must include expiry date and approver.
- Closed risks and validations remain archived for traceability.
